# AI You - Your Personal Chatbot

An AI-powered chatbot that represents {self.name} in other words you, providing information about your background, career, and experience.

## If you want to use
- Please change variable {self.name} to your own name
- Please change push_over credentials and email to your own, DO NOT use mine (lolz)
- In /me folder, add in your own information. Start by downloading your linkedin profile as pdf. use the 3 dots @your-profile-page.
- Rewrite the txt. files to fit in by your life and career, feel free to add more
- This does not have to be a professional career chatbot, change the agent's prompts to be something else
- **When working with AI Agents, the most simple thing to optimize is the prompting, it is the most common problem when an AI system is not going as you want!**

## Features

- Interactive chat interface
- Powered by OpenAI GPT-4-mini (We are not rich..)
- Real-time notifications via Pushover
- Professional, conversational responses

## How to Use

Simply type your message and press Enter to chat with the AI version of yourself. Ask about:
- Professional background and experience
- Career history
- Skills and expertise
- Future goals and aspirations
**You can add in more information (txt.) if you want users to be able to ask more about you**

## Technology Stack

- Flask web framework
- OpenAI GPT-4 API
- Modern responsive UI
- Docker containerization
