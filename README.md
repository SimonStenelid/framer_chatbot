# AI Simon - Personal Assistant

An AI-powered chatbot that represents Simon Stenelid, providing information about his background, career, and experience.

## Features

- Interactive chat interface
- Powered by OpenAI GPT-4
- Real-time notifications via Pushover
- Professional, conversational responses

## How to Use

Simply type your message and press Enter to chat with AI Simon. Ask about:
- Professional background and experience
- Career history
- Skills and expertise
- Future goals and aspirations

## Technology Stack

- Flask web framework
- OpenAI GPT-4 API
- Modern responsive UI
- Docker containerization
